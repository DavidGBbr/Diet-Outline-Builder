# Frontend Components

## Local Diet Form

- Fields: sex, age, heightCM, weightKG, activityLevel, goal.
- Defaults: sample-like values for quick manual testing.
- Validation: Browser-level required/min/max constraints plus API validation from the graph.

## Result Panel

- Shows BMI, classification, total spend, target spend, goal, and macros.
- Shows explanation and safety disclaimer.
- Includes raw JSON for debugging and learning.

## Error Panel

- Shows API validation errors returned by the local server.
